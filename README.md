# golang-basic2
Learning Golang language from Programming Zaman Now Channel

---

### Perintah Terminal
- Inisialisasi Modul / Project
     - `$ go mod init nama_folder`
- Kompilasi File Golang
     - `$ go build main.go`
- Run File 
     - `$ go run main.go`
